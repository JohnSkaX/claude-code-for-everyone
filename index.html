Claude Code for Everyone

Build things with AI. No coding skills needed.

1 Open Terminal
Mac: Press Cmd + Space, type "Terminal", press Enter
Windows: Press Win + X, click "Terminal"
Linux: Press Ctrl + Alt + T
2 Copy & Paste This Command
curl -fsSL https://raw.githubusercontent.com/JohnSkaX/claude-code-for-everyone/main/install.sh | bash
irm https://raw.githubusercontent.com/JohnSkaX/claude-code-for-everyone/main/install.ps1 | iex
3 That's It!
The installer handles everything. When it's done, type claude to start.
No coding experience? That's the point. Just tell Claude what you want in plain English, like "organize my messy Downloads folder" or "create a simple budget tracker."

# Common Tasks with Claude Code

Here are things you can do with Claude Code, even without programming experience.

## File Management

### Organize Files
```
Help me organize my Downloads folder. Move images to a Pictures folder and documents to a Documents folder.
```

### Find Duplicates
```
Find duplicate files in my Photos folder
```

### Rename Files in Bulk
```
Rename all files in this folder to include today's date at the beginning
```

## Text and Documents

### Summarize Long Documents
```
Summarize this document for me: [paste text or provide file path]
```

### Convert Formats
```
Convert this markdown file to HTML
```

### Fix Formatting
```
Clean up the formatting in this text file
```

## Simple Automation

### Daily Checklist
```
Create a script that shows me a daily checklist when I open my terminal
```

### Quick Notes
```
Create a simple system where I can save and view quick notes from the command line
```

### Reminders
```
Create a script that reminds me to take a break every hour
```

## Learning and Exploration

### Explain Code
```
Explain what this code does: [paste code]
```

### Explore Your Computer
```
What files are taking up the most space on my computer?
```

### Learn Concepts
```
Explain how websites work in simple terms
```

## Data Tasks

### Work with Spreadsheets
```
Read this CSV file and tell me the average value in the "price" column
```

### Simple Charts
```
Create a simple bar chart from this data: [provide data]
```

## Tips for Success

1. **Start with "I want to..."** - This frames your request clearly
2. **Provide context** - Tell Claude why you need something
3. **Ask follow-up questions** - "Can you make it simpler?" or "What does this part do?"
4. **Save useful scripts** - Ask Claude to save things you'll use again
